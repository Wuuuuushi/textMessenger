A program that will let the user interact with another user. The default parameter for the server is set to the ICS 32 distributed society. There are still bugs with the loading of recipient selected messages where one of the messages might not load in. Make sure that you load in a profile or make a new profile when using the messenger app, otherwise, it will create an error and nothing will work afterwards. There are some bugs with the messages not loading in properly if the server is running slow.
